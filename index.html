----+-------------------------------------------------
    | ---
    | GitHooks
    | -- A small wrapper on `git` for extra hooks
    | ---
    | Tools
    | -- init
    | ---- Post init hook
    | -- push
    | ---- Pre push hook
    | ---- Post push hook
    | -- pull
    | ---- Post pull hook
    | ---
    | Config
    | -- A config should stored in /etc/gh.conf
    | ---- ~/.config, ~/, $XDG_CONFIG_HOME /will not/ be used
    | -- See example.conf for example
    | -- gh is not designed for a multi-user setup
    | ---- fork gh if you need it.
    | ---
    | Hooks
    | -- Hooks are to be places in $PREFIX/share/gh/$hooktype-$tool
    | ---- ie: /usr/local/share/gh/post-init
